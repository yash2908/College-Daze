HOW TO PLAY:-

a) Fill these boxes with digits 1-9 such that each row and each column has all 9 digits.
b) You should also satisfy the subgrid condition.


Ex:

↠ +16 means all the digits in that subgrid give a total sum of 16
↠ + Indicates sum,
↠ - Indicates difference, [ If -y is above a box then that box - y = other box in the sun grid ]
↠ * Indicates product,
↠ / Indicates quotient, [ If /y is above a box then that box / y = other box in the sun grid ]
